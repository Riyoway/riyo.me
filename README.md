# riyo.me

Riyo's personal website and the front door to the `riyo.me` subdomains.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production

The site is configured as a static Next.js export. `npm run build` writes the
deployable site to `out/`.

Connect this repository to Vercel and assign `riyo.me` and `www.riyo.me` to the
project. Each subdomain is managed as its own project.
